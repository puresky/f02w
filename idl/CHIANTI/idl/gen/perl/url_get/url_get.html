A PERL Program for Retrieving Web Documents

Abstract

url_get allows you to retrieve a document via WWW, gopher, or ftp by specifying the document's URL. The README file has more information.

What's New

url_get now supports proxy servers. See the README file for more information.

Getting url_get

A tarred and compressed version of the program is available, as well as the original PERL scripts, including url_get, url_get.pl, URL.pl, and ftplib.pl.
